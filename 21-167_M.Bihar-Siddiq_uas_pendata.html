Dokumentasi Proyek: Prediksi Heating Load

1. Deskripsi Umum

Proyek ini bertujuan untuk memprediksi Heating Load (Y1) berdasarkan 8 fitur dari dataset Energy Efficiency. Model Machine Learning yang digunakan adalah GradientBoostingRegressor dari Scikit-learn.

2. Tahapan Analisis

2.1 Data Understanding

Dataset memiliki 8 fitur (X1–X8):

  • Relative Compactness
  • Surface Area
  • Wall Area
  • Roof Area
  • Overall Height
  • Orientation
  • Glazing Area
  • Glazing Area Distribution

Target prediksi adalah Y1 (Heating Load).

2.2 Preprocessing

Data diskalakan menggunakan StandardScaler. Tidak ada nilai kosong pada dataset.

2.3 Modeling

Model dievaluasi dengan Mean Squared Error (MSE) dan R² Score. Model terbaik dipilih dan disimpan dalam file model_heating.pkl.

2.4 Deployment

Model di-deploy menggunakan Flask API (app.py) dan form input dari HTML (index.html).

3. Struktur File

energy_predictor/
│
├── index.html         ← Web form input prediksi
├── app.py             ← Backend Flask API
├── scaler.pkl         ← Scaler hasil preprocessing
├── requirements.txt   ← Daftar dependensi

4. Cara Menjalankan

  1. Ekspor model_heating.pkl dan scaler.pkl dari Google Colab
  2. Jalankan Flask API: 
    pip install -r requirements.txt
python app.py
  3. Buka index.html di browser
  4. Masukkan nilai X1–X8 dan klik Prediksi Sekarang

5. Penutup

Web ini memberikan antarmuka sederhana dan intuitif bagi pengguna untuk memprediksi Heating Load dari parameter desain bangunan secara otomatis. Cocok digunakan sebagai demo ML deployment atau tugas akhir.
